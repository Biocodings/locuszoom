Locuszoom

A Javascript/d3 embeddable plugin for interactively visualizing statistical genetic data from customizable sources.

Play With a Demo

Plot Builder
An interactive demo showing LocusZoom plot side by side with its serializable layout object. Use this to try out all the supported elements of a LocusZoom plot.

Top Hits
An example of a responsive LocusZoom plot with some basic form elements to provide navigation to different loci around the genome.
